# Elevate Task-1: Personal Portfolio

A clean and responsive personal portfolio website built with HTML and CSS.

## 🚀 Live Demo

[View Live Demo](https://danetithanuja4-web.github.io/Elevate-Task-1/)

## 📋 Description

This is a modern, responsive portfolio website showcasing web development skills. The site features a clean design with gradient backgrounds, smooth animations, and mobile-friendly layout.

## ✨ Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean interface with gradient backgrounds and smooth transitions
- **Interactive Elements**: Hover effects on navigation and feature boxes
- **Fast Loading**: Optimized HTML and CSS for quick performance
- **Professional Layout**: Header navigation, hero section, feature grid, and footer

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Modern styling with flexbox, grid, and animations
- **Responsive Design**: Media queries for mobile optimization

## 🚀 Getting Started

### Prerequisites

- A web browser (Chrome, Firefox, Safari, etc.)
- No additional dependencies required

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/danetithanuja4-web/Elevate-Task-1.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Elevate-Task-1
   ```

3. Open `Index.html` in your web browser or use a local server.

### Local Development

You can run this project locally by:
- Double-clicking `Index.html` to open in your default browser
- Using a local server like Live Server extension in VS Code
- Using Python's built-in server: `python -m http.server`

## 📱 Sections

- **Header**: Navigation menu with logo

## 🎨 Customization

The CSS uses CSS variables for easy customization. You can modify colors, fonts, and spacing in `Style.css`:

```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  /* Add more variables as needed */
}
```
